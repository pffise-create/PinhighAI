# AWS Deployment Tracker

Use this file to track whether code changes are actually deployed to AWS.

## Status Definitions
- `PENDING`: Implemented in repo, not deployed yet.
- `DEPLOYED`: Code deployed to AWS and verified.
- `ROLLED_BACK`: Deployment reverted.

## Required Evidence For `DEPLOYED`
1. Deploy command/script used.
2. Lambda `LastModified` timestamp from AWS.
3. One smoke-test result (what was tested + pass/fail).

## Commands (Reference)
```bash
# Verify auth/profile before deploy
aws sts get-caller-identity --profile pinhigh-deploy

# Check deployed timestamp for a Lambda
aws lambda get-function-configuration \
  --function-name <lambda-name> \
  --profile pinhigh-deploy \
  --region us-east-1 \
  --query 'LastModified' \
  --output text
```
